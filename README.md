## 🌹 About The Project

This is an interactive romantic proposal website featuring:

✨ Floating heart animations  
🎆 Fireworks explosion when YES is clicked  
🎵 Background romantic music  
💔 Emotional messages when trying to press NO  
😂 A "No" button that runs away  
💖 Beautiful love message reveal  
📱 Fully responsive design
